# v26 Letter Korean Terms + Self-check Update

- Letter PDF의 `Track` 용어를 사용자 표시 영역에서 제거하고 `중점 점검항목`으로 변경했습니다.
- `선택 기간 핵심 메시지`를 `종합 의견`으로 변경했습니다.
- `관리흐름 영향도`를 `품질시스템 영향`으로 변경했습니다.
- `~흐름에 영향` 형태의 문구를 제거하고 GMP/품질시스템 관점의 자연스러운 문장으로 수정했습니다.
- `다음 기간 모니터링 포인트` 영역을 제거하고 `중점 점검항목별 자체점검 질문` 페이지로 대체했습니다.
- `본 페이지는 자동 생성...` 안내문을 제거했습니다.
- 2~3페이지 표 글자 크기를 확대하고, 2페이지와 3페이지를 분리해 가독성을 개선했습니다.
- 2페이지 하단에 `내부 감시용 Checklist 또는 One page Checklist` 확인 안내문을 추가했습니다.
- Letter 구조를 6페이지로 변경했습니다.
  - 1페이지: 핵심 요약 + 종합 의견 + 중점 점검항목 + 선정근거/품질시스템 영향
  - 2페이지: 중점 점검항목별 우선 확인자료
  - 3페이지: 중점 점검항목별 자체점검 질문
  - 4페이지: 상세 트렌드 요약
  - 5페이지: Top5 및 주요 현황
  - 6페이지: 감시분야/등급 차트
- 기간/필터가 변경되면 KPI, 종합 의견, 중점 점검항목, 선정 근거, 품질시스템 영향, 우선 확인자료, 자체점검 질문이 선택 데이터 기준으로 재계산됩니다.
- 연간 선택 시 `연도` 컬럼 문자열/숫자 타입 차이로 빈 데이터가 될 수 있는 부분을 보강했습니다.
