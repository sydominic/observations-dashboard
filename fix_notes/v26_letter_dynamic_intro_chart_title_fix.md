# v26 Letter Dynamic Intro and Chart Title Fix

## 반영 내용
- Letter 2페이지 상단 문구를 고정 기능설명형 문장 대신 선택 기간의 상위 중점 점검항목을 반영하는 반동적 문장으로 변경했습니다.
- 2페이지 문구는 상위 중점 점검항목명과 예상 관리취약점 키워드를 기간별로 조합하여 생성됩니다.
- 기존 `Letter 성격에는 이 방식이 적합합니다.` 및 자동 산정 설명 문구를 제거했습니다.
- 감시분야/등급 차트 페이지의 제목 아래 설명문(`감시분야와 등급 분포를 동일 기간 기준으로 시각화한 결과입니다.`)을 제거했습니다.
- 차트 페이지 제목 위치를 설명문 제거 후의 여백에 맞게 조정했습니다.
- Letter 캐시 토큰을 갱신하여 이전 양식 PDF가 재사용되지 않도록 했습니다.

## 검증
- `python -m py_compile streamlit_app.py` 통과
- 테스트 데이터 기준 2026-1Q, 2026-2Q, 2026-연간 Letter PDF 생성 확인
- PDF 렌더링으로 2페이지 문구 및 5페이지 차트 제목 영역 확인
