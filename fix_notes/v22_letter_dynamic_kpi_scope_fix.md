# v22 Letter 동적 KPI/권장실행방향 및 One page Checklist 오류 수정

## 수정 배경
- Report 탭 하단에서 One page Checklist 선택 시 `UnboundLocalError: scope_suffix` 발생.
- Letter 1페이지의 KPI/핵심 메시지/Track/권장 실행순서가 기간별 데이터 변화를 충분히 반영하지 못함.
- 최다 세부유형이 기존 데이터에 `세부구분` 컬럼이 없으면 `- / 0건`으로 표시되는 문제 발생.

## 수정 내용
1. `scope_suffix`를 사용하기 전에 먼저 계산하도록 순서 수정.
2. Letter 1페이지 최다 세부유형을 기존 컬럼 의존 방식에서 `infer_topic_from_row()` 기반의 종합 세부유형 집계로 변경.
3. 선택 기간별 상위 세부유형 Top3를 계산하여 Track 1~3 제목, 근거, 점검 포인트를 동적으로 생성.
4. 권장 실행순서도 최다 세부유형과 상위 감시분야에 맞춰 동적으로 생성.
5. 핵심 메시지도 선택 기간의 최다 감시분야와 종합 세부유형 Top3를 반영하도록 변경.
6. KPI 카드 문구를 `최다 세부유형(종합)`으로 변경하여 감시분야별이 아닌 전체 기간 집계 기준임을 명확화.
7. “QA 주관” 표현은 계속 배제하고, 각 부서 자체 조치계획 + QA 검토 지원 구조를 유지.

## 검증
- `python -m py_compile streamlit_app.py` 통과.
- 더미 데이터로 2026-1Q, 2026-2Q, 2026-연간 각각 `build_report_pdf()` 실행하여 PDF 생성 확인.
- 생성된 PDF 1페이지 텍스트에서 총 지적, 최다 분야, 최다 세부유형, Track, 권장 실행순서가 기간별로 달라지는 것 확인.
